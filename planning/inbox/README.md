# Planning Inbox

This is the raw intake area for owner notes, screenshots, prompts, and Cursor/Codex reports.

The inbox is not source of truth.

Cursor/Codex may read inbox material and triage it into:

- `docs/00-start-here/NEXT_STEPS.md`
- `docs/00-start-here/CURRENT_STATE.md`
- relevant module docs when a durable decision is made

## Folders

- `notes/` - rough owner notes and bug reports
- `screenshots/` - screenshot references before review
- `prompts/` - raw prompts before they are turned into workflow docs
- `raw-cursor-reports/` - unprocessed reports from Cursor/Codex sessions
- `processed/` - reviewed items, later grouped by `YYYY-MM`

Do not delete inbox items without owner approval.

Do not commit secrets, `.env`, token values, customer private data, or large raw evidence unless explicitly approved.
