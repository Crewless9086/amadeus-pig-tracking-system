# Beacon Campaign Workflow

1. Beacon detects opportunity from backend readiness.
2. Drafts campaign plan/copy/media selection.
3. Owner reviews exact packet.
4. Public post/send/spend remains blocked unless exact gate is approved.
5. Manual or controlled execution evidence is recorded.
6. Performance evidence informs future recommendations.
