# CHARLIE Vault Tables

Normalized Vault tables carry structured mission intelligence.

Markdown Vault Brain files remain the operating manual. Supabase Vault tables remain runtime/searchable memory.

## Target Tables

The normalized layer includes or targets:

- mission projects;
- artifacts;
- agent runs;
- review board findings;
- owner decisions;
- income stream reviews;
- improvement proposals;
- improvement evidence.

Mission metadata can still carry important mission vault fields. Normalized Vault tables are the durable structured layer for cross-mission intelligence and dashboard reporting.

## Relationship To Markdown Brain

- Markdown explains who CHARLIE is, how agents work, what rules apply, and what owner-approved business direction exists.
- Supabase tables record what happened, what agents produced, what the owner decided, and what evidence exists.
- A change to one may require a change to the other. Brain Guard must check this before review-ready.
