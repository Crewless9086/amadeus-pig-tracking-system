# Golden Example Library

Status: seed library active.

Purpose: CHARLIE CORE agents use this folder as examples of what "good" looks like before producing owner-facing artifacts. These examples do not replace mission-specific evidence. They set the quality bar.

Required use:
- UI or dashboard missions must inspect `GOLD_STANDARD_DASHBOARD.md`.
- Mission intake and planning work must inspect `GOLD_STANDARD_MISSION.md`.
- Owner review work must inspect `GOLD_STANDARD_REVIEW_PACKET.md`.
- Focused data-backed product improvements must inspect `GOLD_STANDARD_LITTER_SUMMARY_PR89.md`.
- State-aware UI improvements and Brain Guard recovery work must inspect `GOLD_STANDARD_LITTER_DETAIL_PR90.md`.
- Agent creation or agent upgrades must inspect `GOLD_STANDARD_AGENT_CHARTER.md`.
- Business or income-stream work must inspect `GOLD_STANDARD_BUSINESS_PLAN.md`.
- SAM Meat Sales work must inspect `SAM_MEAT_GOLD_STANDARD_REPLIES.md`.
- Blocked or partial missions must inspect `GOLD_STANDARD_RECOVERY_PACKET.md`.

Maintenance rule: when a mission produces an excellent result, Brain Guard should propose adding a sanitized example here. Owner approval is required before it becomes doctrine.
