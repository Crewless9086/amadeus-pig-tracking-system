# Operations Documentation

## Purpose

This folder owns practical operating, testing, release, and troubleshooting guidance.

## Current Operational Priority

Orders are live and profit-critical. Operational checks should focus first on order lifecycle safety:

- reject
- customer cancel
- reserve
- release
- split requested item sync
- order review for Sam
- web app order usability
